# MEAN-app-PixelArt

Pixel art is a web application that allows users to share their pixel art with other users in the form of images(jpg,png,jpeg ) or gifs .

Technology use : 
***Back End :***
- NodeJS .
- ExpressJs .
- For authorization i used JWT(JsonWebToken) .
- Database : MongoDB .

***Front End :***
- Bootstrap .
- Angular .
